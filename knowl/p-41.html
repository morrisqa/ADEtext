Note that if \(m>0\text{,}\) then since \(y(0)=0\text{,}\) \(y(t)>0\) on \((0,\epsilon)\) for some \(\epsilon>0\text{.}\) Hence, \(|y(t)|=y(t)\text{,}\) and our differential equation simplifies to

\begin{equation*} \left\{ \begin{array}{l} y''=y\\ y(0)=0 \\ y'(0)=m \end{array} \right. \end{equation*}

so long as \(y(t)\ge 0\text{.}\) Similar to ExampleÂ 1.1.1, the solution is \(y(t)=m\sin(t)\) so long as \(0 < t < \pi \text{.}\) When \(t=\pi\text{,}\) we find that \(y(\pi)=0\) and \(y'(\pi)=-m\text{.}\) So we may now consider the initial value problem

\begin{equation*} \left\{ \begin{array}{l} y''=-y\\ y(\pi)=0 \\ y'(\pi)=-m \end{array} \right. \end{equation*}

since \(y(\pi)=0\) and \(y'(\pi)=-m < 0\) implies that \(y(t) < 0 \) for \(t>\pi\text{,}\) so that \(|y(t)|=-y(t)\text{.}\) However, this gives us the solution \(y(t)=-m \sinh (t-\pi)\) for \(t>\pi\text{.}\) This solution remains negative for all \(t>\pi\text{,}\) and hence we need not consider the solution further. Combining our two results gives

\begin{equation*} y(t)=\left\{ \begin{array}{ll} m \sin (t) & 0 < t \le pi, \\ -m \sinh(t-\pi) & t > \pi. \end{array} \right. \end{equation*}

when \(m>0\)
