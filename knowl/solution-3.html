Solution 1.1.3.1.

The solution (which can be found by the same methods used in Example 1.1.1 and Example 1.1.2) is \(y(t) = \frac{1}{\frac{1}{\alpha} - t}\text{.}\) Note that this function is differentiable on \((-\infty,\frac{1}{\alpha}) \cup (\frac{1}{\alpha},\infty)\text{,}\) and since the solution must satisfy the initial value, the solution is only valid on \((-\infty,\frac{1}{\alpha})\text{.}\) Note that when \(\alpha\) is sufficiently large, this means our solution only exist in a small range to the right of \(x=0\text{.}\)
