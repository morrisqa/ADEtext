Initial & Other Values.

Consider the initial value problem
\begin{equation*} \left\{ \begin{array}{l} y''=|y|\\ y(0)=0 \\ y'(0)=m \end{array} \right. \end{equation*}

3.

Find a solution for \(t>0\) each value of \(m\text{.}\)

Hint.

It may be helpful to consider cases where \(m>0\text{,}\) \(m=0\text{,}\) and \(m < 0\text{.}\) This will allow you to simplify \(|y|\) at least near \(t=0\text{.}\)

Answer.

If \(m>0\text{,}\) the solution is

\begin{equation*} y(t)=\left\{ \begin{array}{ll} m \sin (t) & 0 < t \le pi, \\ -m \sinh(t-\pi) & t > \pi. \end{array} \right. \end{equation*}

If \(m < 0\text{,}\) the solution is

\begin{equation*} y(t)= m \sinh (t). \end{equation*}

If \(m=0\text{,}\) the solution is \(y(t)=0\)

Solution.

Note that if \(m>0\text{,}\) then since \(y(0)=0\text{,}\) \(y(t)>0\) on \((0,\epsilon)\) for some \(\epsilon>0\text{.}\) Hence, \(|y(t)|=y(t)\text{,}\) and our differential equation simplifies to

\begin{equation*} \left\{ \begin{array}{l} y''=y\\ y(0)=0 \\ y'(0)=m \end{array} \right. \end{equation*}

so long as \(y(t)\ge 0\text{.}\) Similar to Example 1.1.1, the solution is \(y(t)=m\sin(t)\) so long as \(0 < t < \pi \text{.}\) When \(t=\pi\text{,}\) we find that \(y(\pi)=0\) and \(y'(\pi)=-m\text{.}\) So we may now consider the initial value problem

\begin{equation*} \left\{ \begin{array}{l} y''=-y\\ y(\pi)=0 \\ y'(\pi)=-m \end{array} \right. \end{equation*}

since \(y(\pi)=0\) and \(y'(\pi)=-m < 0\) implies that \(y(t) < 0 \) for \(t>\pi\text{,}\) so that \(|y(t)|=-y(t)\text{.}\) However, this gives us the solution \(y(t)=-m \sinh (t-\pi)\) for \(t>\pi\text{.}\) This solution remains negative for all \(t>\pi\text{,}\) and hence we need not consider the solution further. Combining our two results gives

\begin{equation*} y(t)=\left\{ \begin{array}{ll} m \sin (t) & 0 < t \le pi, \\ -m \sinh(t-\pi) & t > \pi. \end{array} \right. \end{equation*}

when \(m>0\)

If \(m=0\text{,}\) then \(y(t)=0\) is a solution to

\begin{equation*} \left\{ \begin{array}{l} y''=y\\ y(0)=0 \\ y'(0)=m \end{array} \right. \end{equation*}

If \(m < 0\text{,}\) then \(y(0)=0\) and \(y'(0)=m < 0\) imply that \(y(t) < 0\) on \((0,\epsilon)\) for some \(\epsilon>0\text{.}\) Hence, \(|y(t)|=-y(t)\text{,}\) and our differential equation simplifies to

\begin{equation*} \left\{ \begin{array}{l} y''=-y\\ y(0)=0 \\ y'(0)=m \end{array} \right. \end{equation*}

so long as \(y(t)\le 0\text{.}\) But \(y(t)=m \sin(t)\) is a solution which remains negative for all \(t>0\text{.}\) Hence it is the solution for all \(t>0\text{.}\)

By Theorem 1.2.9, each of these solutions is unique.

Lipschitz Functions.

7.

Prove that if \(f\) is differentiable for all \(x \in \mathbb{R}\) and \(f'(x)\) is uniformly bounded, then \(f\) satisfies a Lipschitz condition with respect to \(x\text{.}\)

Solution.

Since \(f'(x)\) is uniformly bounded, there exists some \(K\) such that \(|f'(x)|\le K\) for all \(x\in\mathbb{R}\text{.}\) By Taylor's Theorem, \(|f(x_2)-f(x_1)| = |f'(\xi)| |x_2-x_1| \le K|x_2-x_1|\) which proves that \(f\) is Lipschitz with respect to \(x\text{.}\)

8.

Show that the converse of Exercise 1.3.7 is false by finding a function \(f:\mathbb{R} \to \mathbb{R}\) which satisfies a Lipschitz condition with respect to \(x\) but is not differentiable for all \(x\in\mathbb{R}\text{.}\)

Solution.

Consider \(f(x)=|x|\text{.}\) Then \(|f(x_2)-f(x_1)|= ||x|-|y||\le |x-y|\) by the Reverse Triangle Inequality. So \(f\) satisfies a Lipschitz condition with respect to \(x\) (the Lipschitz constant is \(1\)), but \(f\) is not differentiable at \(x=0\text{.}\)
