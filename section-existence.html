Section 1.1 Motivating Examples

In a first course in differential equations, the question of the existence of a solution is often moot, because one can demonstrate the existence simply by writing down the solution.

Example 1.1.1.

Show that the differential equation

\begin{equation*} \left\{ \begin{array}{l} y''=y \\ y(0)=0 \\ y'(0)=3 \end{array} \right. \end{equation*}

has a solution.

Solution.

Since the differential equation \(y''=y\) is linear with constant coefficients, we could consider its characteristic equation, find the roots, and use those to write down a solution. The initial conditions would then be used to find unknown constants.

For this problem, however, such a method is likely unnecessary. We may observe that we know two linearly independent functions, namely \(y_1(x)=\sin(x)\) and \(y_2(x)=\cos(x)\text{,}\) which satisfy \(y''=y\text{.}\) Hence, any solution must be of the form \(y(t)=c_1y_1(t) + c_2y_2(t)\text{,}\) and the initial conditions are again used to find the unknown constants \(c_1\) and \(c_2\text{.}\)

In this case, the reader should find that \(y(t)=3\sin(t)\) is a solution. What is harder to show is that this is the only solution.

While the existence of a solution can be demonstrated by simply writing down a candidate and showing that it satisfies the initial value problem, it is harder to know whether the solution you have found is the only solution. In a first course in differential equations, you may have enountered such problems.

Example 1.1.2.

Show that the differential equation

\begin{equation*} \left\{ \begin{array}{l} y'=y^{\frac{2}{3}} \\ y(0)=0 \end{array} \right. \end{equation*}

has two solutions.

Solution.

This is a classic example of a separable differential equation. To find the solution, rewrite the differential equation as

\begin{equation} \frac{y'}{y^\frac{2}{3}}=1,\tag{1.1.1} \end{equation}

which is only valid if \(y\neq0\text{.}\) Note that \(y_1(x)=0\) is indeed a solution to the differential equation.

If \(y\neq0\text{,}\) then we may integrate both sides of (1.1.1) with respect to \(x\) from \(0\) to \(t\text{.}\) Then on the left hand side we have

\begin{equation*} \int_0^t \frac{y'}{y^\frac{2}{3}}\; dx= \int_{y_2(0)}^{y_2(t)} \frac{1}{y^\frac{2}{3}} \; dy= \left. 3 (y(t))^\frac{1}{3}\right|_{y_2(0)}^{y_2(t)} = 3(y_2(t))^\frac{1}{3}, \end{equation*}

while on the right hand side, we have

\begin{equation*} \int_0^t 1\; dx = t. \end{equation*}

Hence, our solution is \(3(y_2(t))^\frac{1}{3}=t\text{,}\) or equivalently \(y_2(t)=\frac{1}{27}t^3\text{.}\)

There is also a third, more subtle issue, which deals with the interval on which the solution to a given differential equation is valid. Even for relatively "well-behaved" differential equations, we can run into situations in which the interval of existence is quite small.

Example 1.1.3.

Find the interval of existence for the solution to

\begin{equation*} \left\{ \begin{array}{l} y'=y^2 \\ y(0)=\alpha \end{array} \right. \end{equation*}
Solution.

The solution (which can be found by the same methods used in Example 1.1.1 and Example 1.1.2) is \(y(t) = \frac{1}{\frac{1}{\alpha} - t}\text{.}\) Note that this function is differentiable on \((-\infty,\frac{1}{\alpha}) \cup (\frac{1}{\alpha},\infty)\text{,}\) and since the solution must satisfy the initial value, the solution is only valid on \((-\infty,\frac{1}{\alpha})\text{.}\) Note that when \(\alpha\) is sufficiently large, this means our solution only exist in a small range to the right of \(x=0\text{.}\)

From these three examples, we see that there are four important questions you might ask about an initial value problem:

  1. Does the IVP have a solution on any interval? This is the question of existence.

  2. If the IVP has a solution on a given interval, is there only one solution? This is the question of uniqueness.

  3. If the IVP has a solution which is not unique on a given interval, how many solutions are there? This is the question of multiplicity.

  4. How large can the interval on which the solution to the IVP exists possibly be?

From our examples thus far, however, we have answered these questions only partially. In Example 1.1.1, we proved existence of a solution, but did not establish uniqueness. In Example 1.1.2, we proved nonuniqueness, but did not definitely answer the question of multiplicity (though we did provide a lower bound on the number of solutions). In Example 1.1.3, we showed that the solution could fail to exist arbitrarily close to the initial condition, but did not address how large the interval could be and what causes the solution to fail to exist outside some interval.

In any event, we are in a considerable amount of trouble if our only methods for dealing with these questions rely on writing down explicit solutions. In fact, even simple differential equations of the form

\begin{equation*} y''+a(t)y'+b(t)y=0 \end{equation*}

may not have solutions expressible in terms of elementary (trigonometric, exponential, logarithmic, polynomial, etc.) functions. The interested reader is directed to [1] for an in-depth discussion of when such equations do and do not have elementary solutions.

Moving forward, we wish to address the questions of existence, uniqueness, and multiplicity for initial value problems first, and then subsequently for a new type of problem, called a boundary value problem.
